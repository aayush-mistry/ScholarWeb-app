# ScholarWeb

ScholarWeb is an AI-powered educational platform designed to help students discover, organize, and optimize their learning journey through personalized recommendations, intelligent resource discovery, and interactive knowledge visualization.

## Problem Statement

Students often face challenges such as scattered learning resources, lack of personalized guidance, ineffective progress tracking, and difficulty understanding relationships between concepts. ScholarWeb addresses these challenges by creating a centralized learning ecosystem powered by AI and data-driven insights.

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd ScholarWeb
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

### Environment Variables

Create a `.env` file in the server directory and configure the required environment variables.

### Run Frontend

```bash
cd client
npm run dev
```

### Run Backend

```bash
cd server
npm run dev
```
